# 《云水江湖》游戏核心玩法机制文档

> 文档版本：v1.0  
> 项目名称：云水江湖  
> 英文名建议：Inkblade: Tales of Jianghu  
> 游戏类型：水墨国风武侠卡牌构筑 Roguelike  
> 核心参考：卡牌构筑、回合制战斗、路线选择、Roguelike 重复游玩  
> 文档用途：用于玩法系统设计、程序架构拆分、数值框架、关卡设计、战斗原型开发与后续产品迭代。

---

## 1. 游戏核心定位

《云水江湖》是一款以水墨国风武侠为主题的卡牌构筑 Roguelike 游戏。

玩家选择不同历史幻想角色进入“云水江湖”，从一副较弱的初始牌组开始，通过战斗、事件、商店、休息、法宝、心法与路线选择不断成长，最终深入墨灾源头，面对角色自身的心魔与改写命运的诱惑。

游戏核心体验可以概括为：

> **用卡牌打出武侠招式，用构筑形成角色流派，用心境和墨灾承载叙事选择。**

---

## 2. 玩法设计目标

### 2.1 核心设计目标

1. **保留卡牌 Roguelike 的构筑爽感**
   - 初始牌组弱。
   - 每局逐步成长。
   - 每次奖励选择都会影响后续构筑。
   - 每一局都有不同组合。

2. **建立明确的国风武侠差异化**
   - 卡牌不是技能按钮，而是“招式”。
   - 遗物不是装备，而是“法宝”。
   - 路线不是爬塔，而是“江湖行旅”。
   - 状态不是普通 Buff，而是“心境”和“墨灾”。

3. **让角色玩法与人物气质强绑定**
   - 貂蝉：魅惑、舞势、离间、闪避。
   - 蔡文姬：琴音、余韵、净化、终曲。
   - 赵云：枪势、破阵、护主、孤勇。
   - 诸葛亮：筹策、观星、阵法、借风。

4. **让叙事选择进入玩法系统**
   - 心境影响战斗与事件。
   - 墨灾牌带来强力效果与长期代价。
   - 角色结局由战斗倾向与事件选择共同决定。

| 支柱 | 说明 |
|---|---|
| 卡牌构筑 | 通过战斗奖励、商店、事件不断优化牌组 |

| 江湖路线 | 玩家在地图节点中选择风险与收益 |

| 心境变化 | 战斗和事件影响角色精神状态 |
| 墨灾代价 | 强力黑暗牌提供高收益与高风险 |
| 法宝心法 | 长期被动能力改变构筑方向 |
| 多局成长 | 死亡不是终点，而是解锁信息与内容的过程 |

## 4. 单局游戏核心循环

### 4.1 单局流程概览

```text
选择角色
  ↓
进入第一章地图
  ↓
选择路线节点
  ↓
进入战斗 / 事件 / 商店 / 休息 / 宝箱
  ↓
获得奖励或承担代价
  ↓
调整牌组、生命、法宝、心境
  ↓
击败章节 Boss
  ↓
进入下一章
  ↓
最终 Boss / 心魔结局
  ↓
通关或死亡结算
```

---

### 4.2 玩家每次决策的核心问题

| 场景 | 玩家问题 |
|---|---|
| 战斗中 | 这一回合我该攻击、防御、铺垫还是爆发？ |
| 战斗奖励 | 我该拿哪张牌？还是跳过？ |
| 地图路线 | 我要走安全路线还是挑战精英？ |
| 商店 | 我该买牌、买法宝、删牌还是回血？ |
| 事件 | 我要安全收益，还是承担墨灾风险？ |
| 休息点 | 我要回血、升级牌，还是修炼心法？ |
| Boss 前 | 我的牌组是否能应对当前章节机制？ |

## 5. 游戏模式

## 5.1 标准模式

标准 Roguelike 单局模式。

特点：

- 每局从初始牌组开始。
- 地图随机生成。
- 战斗、奖励、事件随机出现。
- 死亡后本局牌组清空。
- 可保留图鉴、角色剧情、部分解锁内容。

---

## 5.2 剧情推进模式

以角色剧情和世界观解锁为核心。

特点：

- 每名角色有专属剧情线。
- 首次击败章节 Boss 后解锁对应剧情片段。
- 不同心境和墨灾倾向会影响结局。
- 可用于新手引导和角色理解。

---

## 5.3 挑战模式

后期可扩展。

示例：

| 模式 | 规则 |
|---|---|
| 墨灾深重 | 开局获得强力墨灾牌，但每章墨痕上限降低 |
| 无法休息 | 休息点不能回血，只能升级或修炼 |
| 单一心境 | 全局只能进入一种心境 |
| 法宝乱流 | 每场战斗随机获得一个临时法宝 |
| 无名江湖 | 只能使用通用牌和法宝 |

## 6. 战斗系统总览

## 6.1 战斗基础流程

每场战斗由多个回合组成。

单回合流程：

```text
回合开始
  ↓
触发回合开始效果
  ↓
抽牌
  ↓
获得能量
  ↓
显示敌人意图
  ↓
玩家打牌
  ↓
玩家结束回合
  ↓
弃置未保留手牌
  ↓
敌人行动
  ↓
护甲清空
  ↓
进入下一回合
```

---

## 6.2 战斗区域

| 区域 | 说明 |
|---|---|
| 抽牌堆 | 尚未抽到的牌 |
| 手牌 | 当前回合可使用的牌 |
| 弃牌堆 | 使用后或回合结束弃置的牌 |
| 消耗堆 | 本场战斗移除的牌 |
| 临时生成区 | 战斗中生成、战后消失的牌 |
| 敌人区 | 敌人单位、意图、状态 |
| 角色状态区 | 生命、护甲、能量、资源、心境、墨痕 |
| 法宝区 | 当前拥有的被动法宝效果 |

---

## 6.3 抽牌与洗牌规则

基础规则：

```text
每回合开始抽5张牌。
若抽牌堆不足，则先抽完剩余牌，再将弃牌堆洗入抽牌堆继续抽。
消耗牌不会进入弃牌堆。
保留牌不会在回合结束弃置。
临时牌在战斗结束后移除。
```

设计目的：

- 保证卡牌循环。
- 让牌组大小影响关键牌出现概率。
- 让删牌、抽牌、检索、保留成为重要构筑方向。

---

## 6.4 能量规则

基础规则：

```text
玩家每回合获得3点能量。
能量用于支付卡牌费用。
未使用能量默认在回合结束清空。
部分角色、法宝、卡牌可保留或额外获得能量。
```

可扩展机制：

| 机制 | 说明 |
|---|---|
| 临时能量 | 只在本回合可用 |
| 下回合能量 | 延迟到下回合获得 |
| 能量上限变化 | Boss 法宝或诅咒可改变 |
| X 费卡牌 | 消耗当前所有能量计算效果 |

---

## 6.5 护甲与伤害规则

```text
护甲抵挡攻击伤害。
护甲默认在玩家回合开始或结束时清空，具体以系统规则统一。
建议采用：玩家回合开始时清空上回合护甲。
生命归零则战斗失败。
```

建议采用较清晰的规则：

```text
玩家打出护甲牌 → 敌人行动时抵挡伤害 → 下一回合开始清空剩余护甲。
```

---

## 6.6 敌人意图

敌人每回合提前显示下一次行动意图。

| 意图类型 | 表现 |
|---|---|
| 攻击 | 显示预计伤害 |
| 多段攻击 | 显示攻击次数和单次伤害 |
| 防御 | 敌人获得护甲 |
| 强化 | 敌人获得增益 |
| 弱化 | 玩家获得负面状态 |
| 召唤 | 召唤小怪 |
| 蓄力 | 下回合释放强力技能 |
| 特殊 | Boss 机制或剧情行动 |

国风包装建议：

| 通用意图 | 国风表现 |
|---|---|
| 攻击 | 杀意 |
| 防御 | 运功 |
| 强化 | 蓄势 |
| 弱化 | 施咒 |
| 召唤 | 号令 |
| 蓄力 | 聚煞 |
| 逃跑 | 遁走 |

## 7. 卡牌系统

## 7.1 卡牌基础字段

每张卡牌建议拥有以下数据字段：

```json
{
  "id": "card_id",
  "name": "卡牌名",
  "rarity": "common/uncommon/rare/event/curse/status",
  "type": ["attack", "skill"],
  "cost": 1,
  "target": "enemy/self/all/random",
  "effect": [],
  "keywords": [],
  "character": "common/diaochan/caiwenji/zhaoyun/zhugeliang",
  "upgrade_id": "card_id_plus",
  "flavor": "风味文本"
}
```

---

## 7.2 卡牌类型

| 类型 | 说明 |
|---|---|
| 攻击 | 造成伤害为主 |
| 技能 | 防御、抽牌、状态、资源 |
| 身法 | 闪避、保留、移动、节奏 |
| 琴音 | 蔡文姬专属，产生音律或余韵 |
| 阵法 | 诸葛亮专属，产生持续战场效果 |
| 心境 | 改变或利用心境 |
| 能力 | 本场战斗持续生效 |
| 墨灾 | 高收益高风险 |
| 状态 | 战斗内负面牌 |
| 诅咒 | 跨战斗负面牌 |

---

## 7.3 稀有度

| 稀有度 | 说明 |
|---|---|
| 普通 | 基础补强，容易理解 |
| 罕见 | 构筑组件，具有联动 |
| 稀有 | 构筑核心或强力转折 |
| 事件 | 特殊事件奖励 |
| 状态 | 临时负面 |
| 诅咒 | 长期负面 |

---

## 7.4 卡牌升级

卡牌可以在休息点、事件、法宝或战斗奖励中升级。

升级方向：

| 类型 | 示例 |
|---|---|
| 数值提升 | 伤害+3，护甲+3 |
| 费用降低 | 2费变1费 |
| 增加关键词 | 获得保留、消耗、抽牌 |
| 降低代价 | 墨痕减少 |
| 增强条件收益 | 若满足条件，额外效果提高 |
| 改变类型 | 普通技能升级后附带心境标签 |

升级设计原则：

- 普通牌升级应直接。
- 罕见牌升级应强化构筑联动。
- 稀有牌升级可改变打法。
- 墨灾牌升级通常减少代价或放大收益，但不能两者都过强。

---

## 7.5 卡牌获得

| 来源 | 内容 |
|---|---|
| 普通战斗 | 3选1卡牌奖励 |
| 精英战斗 | 更高概率出现稀有牌 |
| Boss 战 | 稀有牌、心法、法宝 |
| 商店 | 购买卡牌或删牌 |
| 事件 | 特殊牌、墨灾牌、心境牌 |
| 宝箱 | 法宝为主，少量特殊牌 |
| 心魔节点 | 高风险强力牌 |

---

## 7.6 跳过拿牌

玩家可以选择不拿战斗奖励牌。

设计意义：

- 防止牌组臃肿。
- 强化牌组构筑意识。
- 保留高水平策略空间。

建议：

```text
跳过普通奖励可获得少量金币或无奖励。
部分法宝可强化“跳过奖励”的收益。
```

## 8. 角色机制系统

四名首发角色应拥有清晰差异。

| 角色 | 核心资源 | 核心机制 | 定位 |
|---|---|---|---|
| 貂蝉 | 舞势 | 魅惑、舞姿、闪避、离间 | 控制爆发 |
| 蔡文姬 | 音律 | 余韵、五音、净化、杂音 | 防御节奏 |
| 赵云 | 枪势 | 破阵、护主、孤勇 | 稳定攻防 |
| 诸葛亮 | 筹策 | 观星、阵法、借风 | 高阶控场 |

---

## 8.1 貂蝉机制

### 舞势

```text
舞势上限：8
战斗开始：0
通过身法、魅惑、闪避和招式链获得
用于强化攻击、触发连舞、获得闪避或抽牌
回合结束保留一半，向下取整
```

### 魅惑

```text
每层魅惑降低敌人5%攻击伤害。
达到5层时，敌人下次攻击有概率偏转或自伤。
对Boss不完全控制，只降低伤害或触发特殊弱化。
```

### 舞姿

| 舞姿 | 效果 |
|---|---|
| 惊鸿 | 第一次身法牌费用-1 |
| 闭月 | 魅惑效果提高 |
| 破阵 | 舞势强化攻击伤害 |

### 设计关键词

**优雅、控制、闪避、离间、爆发。**

---

## 8.2 蔡文姬机制

### 音律

```text
音律上限：10
战斗开始：0
通过琴音、净化、余韵和五音获得
用于强化琴音、释放终曲和抵消负面效果
回合结束全部保留
```

### 余韵

```text
带有余韵的卡牌，会在下回合开始再次触发部分效果。
同一时间最多保存3个余韵。
```

### 五音

```text
宫、商、角、徵、羽。
集齐五音后获得1层终曲。
终曲用于强化稀有牌或释放爆发。
```

### 杂音

```text
杂音是蔡文姬相关负面状态牌。
抽到后干扰手牌或消耗音律。
蔡文姬可以通过净化将杂音转化为收益。
```

### 设计关键词

**清雅、节奏、防御、净化、终曲。**

---

## 8.3 赵云机制

### 枪势

```text
枪势上限：6
战斗开始：0
通过攻击、防御、破甲和招式链获得
用于强化攻击、防御和护主
回合结束全部保留
若一回合未打出攻击牌，失去1枪势
```

### 破阵

```text
一回合内打出第3张攻击牌时触发破阵。
破阵造成额外伤害，并获得1枪势。
部分卡牌在破阵后升级效果。
```

### 护主

```text
护主：下次受到攻击时减少固定伤害。
若成功抵消全部伤害，获得1枪势。
```

### 孤勇

```text
生命低于50%时，第一次破阵额外造成伤害。
生命低于25%时，每场战斗一次获得枪势和护主。
```

### 设计关键词

**稳定、英勇、破阵、守护、逆转。**

---

## 8.4 诸葛亮机制

### 筹策

```text
筹策上限：9
战斗开始：1
通过观星、布阵、无伤回合和反制敌意获得
用于调整牌序、强化阵法、借风爆发和抵消伤害
回合结束全部保留
```

### 观星

```text
观星X：查看抽牌堆顶X张牌，可以将任意数量置于牌堆底，其余按任意顺序放回牌堆顶。
```

### 阵法

同一时间默认只能存在一个主阵法。

| 阵法 | 效果 |
|---|---|
| 八阵 | 回合结束获得护甲 |
| 火阵 | 状态伤害追加火伤 |
| 风阵 | 每回合第一次技能牌抽1张 |
| 石阵 | 敌人多段攻击降低 |
| 空城阵 | 若不打攻击牌，敌人攻击降低 |

### 借风

```text
当筹策达到6点以上时，可使用借风类卡牌。
借风通常消耗筹策，产生全体伤害、抽牌、回能或阵法强化。
```

### 设计关键词

**预判、控牌、布阵、借势、延迟爆发。**

## 9. 通用机制系统

## 9.1 心境系统

心境是连接玩法与叙事的核心系统。

### 心境基础规则

```text
角色同一时间只能拥有一种主心境。
进入新心境时替换旧心境。
心境影响部分卡牌效果、事件选项和结局倾向。
```

### 心境类型

| 心境 | 玩法方向 | 叙事含义 |
|---|---|---|
| 宁 | 防御、净化、稳定 | 心如止水 |
| 怒 | 攻击、爆发、代价 | 杀意上涌 |
| 悲 | 受伤收益、反击 | 记忆与失去 |
| 魅 | 控制、闪避、扰乱 | 操纵人心 |
| 乱 | 抽牌、随机、墨灾 | 心神失序 |
| 悟 | 转化、消耗、终局 | 看破虚妄 |

---

### 9.1.1 心境倾向值

除战斗中的当前心境外，角色还应记录整局倾向值。

```text
宁值、怒值、悲值、魅值、乱值、悟值
```

来源：

- 使用心境牌。
- 事件选择。
- 墨灾牌使用。
- 击败特定 Boss。
- 角色剧情选择。

用途：

- 解锁事件选项。
- 决定角色结局。
- 影响部分法宝和心法。
- 触发心魔事件。

---

## 9.2 墨灾系统

墨灾是高风险高收益系统。

### 墨痕

```text
墨痕是玩家身上的负面累积。
每有1层墨痕，战斗结束后失去1点生命。
若墨痕达到5层，随机加入1张“墨蚀”到抽牌堆。
```

### 墨灾牌

特点：

- 效果强。
- 费用低或爆发高。
- 会获得墨痕或加入状态牌。
- 可能影响结局。
- 常来自事件、墨灾节点、特殊敌人和墨书阁。

### 墨灾风险等级

| 墨痕层数 | 状态 |
|---:|---|
| 0-2 | 轻微污染 |
| 3-4 | 墨色缠身 |
| 5-7 | 心神受侵 |
| 8+ | 心魔临界 |

建议：

```text
战斗结束时墨痕清空，但结算生命损失和心境倾向。
部分诅咒或事件可让墨痕跨战斗残留。
```

## 9.3 招式链系统

招式链是本作区别于普通卡牌游戏的重要武侠创新。

### 基础规则

玩家一回合内连续打出不同类型卡牌，会形成招式链。

示例：

```text
身法 → 攻击 → 技能 = 游龙回身
攻击 → 攻击 → 攻击 = 连斩
技能 → 技能 → 攻击 = 蓄势一击
心境 → 攻击 → 攻击 = 由心入刃
```

---

### 招式链设计目标

1. 让玩家觉得自己在“出招”。
2. 鼓励规划出牌顺序。
3. 为角色专属资源提供额外获得方式。
4. 提供跨角色通用策略空间。
5. 增强武侠表现力。

---

### 初版推荐招式链

| 顺序 | 名称 | 效果 |
|---|---|---|
| 攻击 → 攻击 → 攻击 | 连斩 | 对当前目标造成4点额外伤害 |
| 技能 → 攻击 | 蓄势 | 该攻击额外造成3点伤害 |
| 身法 → 攻击 | 追影 | 获得2点护甲并造成2点额外伤害 |
| 心境 → 技能 | 静守 | 获得3点护甲 |
| 心境 → 攻击 | 心刃 | 造成3点额外伤害 |
| 技能 → 技能 → 技能 | 固守 | 获得6点护甲 |
| 墨灾 → 攻击 | 墨袭 | 造成5点额外伤害，获得1层墨痕 |
| 消耗 → 攻击 | 断招 | 抽1张牌 |

---

### 与角色资源联动

| 角色 | 联动 |
|---|---|
| 貂蝉 | 身法链获得舞势 |
| 蔡文姬 | 琴音链获得音律或余韵 |
| 赵云 | 连续攻击链触发破阵 |
| 诸葛亮 | 技能/阵法链获得筹策 |

## 9.4 法宝系统

法宝是本作的遗物系统。

### 基础规则

```text
法宝为永久被动效果。
一局内持续生效。
可由精英、Boss、宝箱、事件、商店获得。
部分法宝有负面代价。
```

### 法宝类型

| 类型 | 说明 |
|---|---|
| 通用法宝 | 所有角色可用 |
| 角色法宝 | 只强化某个角色机制 |
| 墨灾法宝 | 强力但带污染 |
| 心境法宝 | 与心境变化相关 |
| 事件法宝 | 由特定事件获得 |
| Boss 法宝 | 击败章节 Boss 后获得 |

---

## 9.5 心法系统

心法是介于法宝和角色流派之间的成长系统。

### 设计定位

法宝偏随机被动。  
心法偏角色长期流派选择。

### 获取方式

- 章节 Boss 奖励。
- 角色事件。
- 特殊修炼节点。
- 精英战特殊奖励。
- 心魔战后选择。

### 心法特点

- 数量少但影响大。
- 通常与角色专属机制强绑定。
- 一局内可获得多个，但建议限制主心法数量。
- 可作为构筑方向转折点。

### 示例

| 角色 | 心法 | 作用 |
|---|---|---|
| 貂蝉 | 倾城心法 | 强化魅惑 |
| 蔡文姬 | 五音归一 | 强化终曲 |
| 赵云 | 龙胆心法 | 强化枪势 |
| 诸葛亮 | 八阵心解 | 强化阵法 |

## 13.4 删牌

删牌是构筑稳定性的关键系统。

规则建议：


设计目的：

- 强化“少而精”构筑意识。
- 让基础牌在后期逐渐被替换。
- 为商店提供长期价值。

## 19. 新手引导设计

## 19.1 教学目标

新手前 10 分钟应理解：

1. 每回合抽牌。
2. 能量限制出牌。
3. 敌人意图需要应对。
4. 护甲抵挡伤害。
5. 战斗后选牌。
6. 地图路线选择。
7. 牌组不是越大越好。
8. 每名角色有专属资源。

---

## 19.2 推荐新手角色

建议将 **赵云** 作为默认推荐角色。

原因：

- 初始生命高。
- 攻防稳定。
- 枪势和破阵容易理解。
- 不依赖复杂控牌或状态转换。
- 具有明确武侠爽感。

---

## 19.3 第一场战斗教学

第一场战斗应只出现：

- 单个敌人。
- 明确攻击意图。
- 玩家有攻击和防御牌。
- 敌人伤害不高。
- 战斗后给出简单奖励。

教学提示：

```text
敌人准备攻击你。
打出防御牌可以获得护甲，抵挡伤害。
如果敌人生命很低，也可以直接攻击击败它。
```

## 22. 平衡风险

## 22.1 角色机制过复杂

风险：  
四名角色都有专属资源，新手难以上手。

解决：

- 默认推荐赵云。
- 每名角色第一局有轻教学。
- 前期只开放少量关键词。
- 卡牌描述保持短句。
- 高阶机制通过稀有牌逐步出现。

---

## 22.2 通用牌抢专属机制

风险：  
通用牌太强，角色差异减弱。

解决：

- 通用牌只能提供“弱版本”专属能力。
- 专属牌在角色机制上效率更高。
- 后期奖励偏向专属牌。
- 部分关键词只允许专属牌高频使用。

---

## 22.3 墨灾代价不足

风险：  
墨灾牌变成无脑强牌。

解决：

- 墨痕必须影响战斗后生命。
- 墨灾牌可能加入状态牌。
- 事件与结局记录墨灾倾向。
- 强力墨灾牌多为消耗或稀有。
- 墨灾流需要特定法宝才能稳定。

---

## 22.4 心境系统沦为装饰

风险：  
心境只影响剧情，不影响玩法。

解决：

- 心境牌进入普通奖励池。
- 每种心境有明确战斗收益。
- 法宝和心法与心境联动。
- 事件选项受心境影响。
- 结局由心境倾向决定。

---

## 22.5 战斗节奏过慢

风险：  
国风表现和复杂机制让战斗拖沓。

解决：

- 普通战斗控制在 3-5 回合。
- 精英战斗控制在 6-9 回合。
- Boss 战控制在 10-15 回合。
- 动画可加速。
- 招式链反馈简洁，不打断过久。

2. “我能看出敌人下一步想干什么。”
3. “我想再来一局试试不同构筑。”

6. “墨灾牌很诱人，但我会担心代价。”
7. “心境和角色故事是有关联的。”
8. “我能明显感受到牌组逐渐成型。”

## 26. 总结

《云水江湖》的核心玩法不是简单复刻传统卡牌爬塔，而是将以下系统融合：

```text
卡牌构筑
  +
回合制战斗
  +
江湖路线选择
  +
角色专属资源
  +
心境变化
  +
墨灾代价
  +
武侠招式链
  +
碎片化叙事
```

最终目标是让玩家在每一局中体验到：

> **从一副普通牌组开始，逐渐练成一套属于自己的江湖武学。**

玩家不是单纯在打牌，而是在：

- 选择自己的心境；
- 承担墨灾的诱惑；
- 形成独特的招式链；
- 面对角色无法逃避的过去；
- 在被书写的命运中夺回自己的笔。
