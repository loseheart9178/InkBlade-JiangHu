# 《云水江湖》通用牌组设计文档

> 文档版本：v1.0  
> 适用范围：通用卡牌池、战斗系统、事件奖励、商店、敌人掉落、构筑平衡  
> 对应项目：水墨国风武侠卡牌 Roguelike《云水江湖》

---

## 1. 通用牌组定位

通用牌组不是某个角色的专属武学，而是所有可选角色都可能在江湖中习得、拾得、领悟或被墨灾污染后获得的公共卡牌池。

玩家一局游戏中的完整牌组通常由三部分组成：

```text
角色初始牌组
  +
角色专属牌池
  +
通用牌池
```

三者分工如下：

| 牌池 | 作用 |
|---|---|
| 角色初始牌组 | 决定角色开局手感和基础操作 |
| 角色专属牌池 | 决定角色核心机制、身份特色和主要流派 |
| 通用牌池 | 提供补强、转型、泛用构筑、事件变化和风险收益 |

通用牌组的目标不是让四名角色变得相同，而是提供一套“江湖共同语言”。同一张通用牌在不同角色手里会产生不同价值。

例如：

- “静心”对蔡文姬是清音防御组件；
- 对赵云是稳定守护节奏；
- 对诸葛亮是等待布局的手段；
- 对貂蝉则可能是从魅惑杀局中短暂抽身。

---

## 2. 通用牌组设计原则

### 2.1 不抢角色专属机制

通用牌可以提供基础攻击、防御、过牌、状态、心境、墨灾风险等功能，但不能完全替代角色专属机制。

| 专属角色 | 不能被通用牌抢走的能力 |
|---|---|
| 貂蝉 | 高效率魅惑、舞势连击、离间控制 |
| 蔡文姬 | 高效率音律、余韵、净化转化、五音终曲 |
| 赵云 | 枪势、破阵、护主、孤勇逆转 |
| 诸葛亮 | 高强度观星、阵法、筹策、借风爆发 |

通用牌只能提供“少量相关功能”或“构筑补强”，不能成为更强的替代品。

---

### 2.2 补足构筑短板

通用牌的核心价值是帮助玩家补短板。

| 角色 | 常见短板 | 通用牌补强方向 |
|---|---|---|
| 貂蝉 | 生存不稳、对单体 Boss 控制收益下降 | 护甲、保留、基础爆发、少量净化 |
| 蔡文姬 | 前期伤害偏慢、启动依赖节奏 | 攻击、抽牌、保留、状态利用 |
| 赵云 | 控制能力弱、容易被状态牌干扰 | 虚弱、净化、抽牌、心境转换 |
| 诸葛亮 | 血量低、前期直接输出弱 | 基础攻击、保命、低费防御、保留 |

---

### 2.3 支持风险收益

通用牌应包含部分“墨灾牌”和“心境牌”，让玩家面对诱惑：

> 这张牌很强，但我是否愿意承担墨痕、污染、生命损失或结局偏移的代价？

风险收益牌是《云水江湖》区别于普通武侠卡牌的重要部分。

---

### 2.4 保持国风武侠语感

通用卡牌命名应避免过度现代化，应来自武学、江湖、山水、心境和墨灾意象。

推荐词源：

| 方向 | 示例 |
|---|---|
| 基础武学 | 劈风、断竹、格挡、卸力、藏锋 |
| 江湖手段 | 暗镖、金针、药酒、听风、传闻 |
| 身法轻功 | 轻身、燕掠、踏水、云步、回身 |
| 心境变化 | 静心、怒斩、悲鸣、乱心、照心 |
| 墨灾力量 | 墨刃、黑雨、纸人、墨龙、未写之页 |
| 通用心法 | 守拙、含光、心如止水、大道至简 |

## 3. 通用卡牌类型

| 类型 | 说明 |
|---|---|
| 攻击 | 直接造成伤害，部分附带易伤、虚弱或多段效果 |
| 技能 | 防御、抽牌、状态处理、资源操作 |
| 身法 | 闪避、保留、跨回合收益、轻量抽牌 |
| 心境 | 改变或利用角色心境 |
| 能力 | 本场战斗持续生效，通常是构筑核心 |
| 墨灾 | 高收益高风险牌，通常带墨痕或污染代价 |
| 状态 | 临时负面牌，多由敌人或墨灾效果加入 |
| 诅咒 | 长期负面牌，多由事件、墨书或强力代价产生 |

## 4. 通用关键词

### 4.1 护甲

```text
护甲：抵挡本回合受到的攻击伤害，回合结束时清空。
```

### 4.2 闪避

```text
闪避：抵消下一次敌人的攻击伤害。若敌人为多段攻击，只抵消其中一段。
```

设计备注：闪避是强力关键词，通用牌池中数量必须严格控制，避免削弱貂蝉特色。

### 4.3 保留

```text
保留：回合结束时不会被弃置，继续留在手牌中。
```

### 4.4 消耗

```text
消耗：使用后，本场战斗内移出牌组。
```

### 4.5 临时

```text
临时：战斗结束后从牌组中移除，不会永久加入玩家牌组。
```

### 4.6 易伤

```text
易伤：受到的攻击伤害提高。建议初版：每层提高25%，回合结束减少1层。
```

### 4.7 虚弱

```text
虚弱：造成的攻击伤害降低。建议初版：每层降低25%，回合结束减少1层。
```

### 4.8 墨痕

```text
墨痕：负面状态。每有1层墨痕，战斗结束后失去1点生命。若墨痕达到5层，随机加入1张“墨蚀”到抽牌堆。
```

墨痕是通用墨灾流的核心风险资源。

### 4.9 心境

角色同一时间只能处于一种主心境。进入新心境时，旧心境会被替换。

| 心境 | 玩法方向 |
|---|---|
| 宁 | 防御、净化、稳定 |
| 怒 | 攻击、爆发、代价 |
| 悲 | 受伤收益、反击、亡魂 |
| 魅 | 控制、扰乱、闪避 |
| 乱 | 抽牌、随机、墨灾力量 |
| 悟 | 转化、消耗、删牌、终局收益 |

## 5. 稀有度与数值基准

### 5.1 稀有度定义

| 稀有度 | 定位 |
|---|---|
| 普通 | 基础补强，效果直观，适合前期 |
| 罕见 | 构筑组件，有明确联动 |
| 稀有 | 构筑核心，改变打法 |
| 事件 | 只能通过事件获得，强叙事或特殊效果 |
| 墨灾 | 高收益高风险，常与墨痕、污染、心魔结局相关 |

### 5.2 费用效果基准

| 费用 | 攻击基准 | 防御基准 | 抽牌基准 |
|---|---:|---:|---:|
| 0费 | 2-5伤害 | 2-4护甲 | 通常带消耗或条件 |
| 1费 | 6-8伤害 | 5-8护甲 | 抽1张，或有条件抽2张 |
| 2费 | 12-16伤害 | 10-14护甲 | 抽2-3张，通常有代价 |
| 3费 | 22-30伤害 | 18-24护甲 | 稀有强力效果 |

## 11. 墨灾牌体系

### 11.1 墨灾牌设计目标

墨灾牌是强诱惑、高代价卡牌。它们适合在以下场景出现：

- 玩家陷入危局，需要翻盘手段；
- 事件提供强力但危险的奖励；
- 玩家主动走“乱”“心魔”“墨书”路线；
- 后期结局倾向需要被战斗选择影响。

墨灾牌必须带来真实代价，代价可以包括：

| 代价 | 表现 |
|---|---|
| 墨痕 | 战后失血、污染牌堆 |
| 状态牌 | 加入墨蚀、墨滞、墨影 |
| 生命损失 | 当前或战后扣血 |
| 心境偏移 | 进入乱、悲或心魔倾向 |
| 结局偏移 | 增加心魔结局权重 |
| 牌组污染 | 加入诅咒牌或临时负面牌 |

| 章节 | 通用牌比例 | 角色专属牌比例 |
|---|---:|---:|
| 第一章 | 45% | 55% |
| 第二章 | 35% | 65% |
| 第三章 | 30% | 70% |

设计原因：前期需要通用牌补足基础，中后期应鼓励角色专属构筑成型。

---

### 17.2 商店投放

| 类型 | 建议比例 |
|---|---:|
| 普通通用牌 | 35% |
| 罕见通用牌 | 25% |
| 角色专属牌 | 30% |
| 特殊/事件牌 | 10% |

商店承担“补短板”和“找通用组件”的作用。

---

### 17.3 事件奖励

事件更适合提供：

- 心境牌；
- 墨灾牌；
- 事件专属牌；
- 强力但带代价的牌；
- 与剧情选择绑定的牌。

---

### 17.4 Boss 奖励

Boss 奖励中不建议出现普通通用牌。建议奖励池包括：

- 稀有角色专属牌；
- 稀有通用牌；
- 强力遗物；
- 心法；
- 高风险墨灾牌。

## 21. 平衡风险与注意事项

### 21.1 通用牌不能过强

如果通用牌过强，会导致角色专属机制失去意义。尤其注意：

- 通用抽牌不能强于诸葛亮控牌；
- 通用净化不能强于蔡文姬；
- 通用闪避不能强于貂蝉；
- 通用连击不能强于赵云。

### 21.2 墨灾牌要有真实代价

墨灾牌如果代价过轻，会成为无脑强牌。代价至少应影响以下之一：

- 当前战斗风险；
- 战斗后生命；
- 牌组污染；
- 心境倾向；
- 结局倾向；
- 后续事件选项。

### 21.3 心境牌要避免无成本频繁切换

如果心境切换过于廉价，心境系统会变成单纯触发器。建议：

- 0费心境牌大多应消耗；
- 强力心境切换应有代价；
- 心境能力牌应限制每回合触发次数。

### 21.4 回复牌必须严格控制

Roguelike 中生命是长期资源。通用回复牌不能过多，否则路线选择和休息点价值会下降。

建议：

- 回复牌多为消耗；
- 回复数值偏低；
- 强回复绑定事件或代价；
- 不提供稳定无限回复循环。

## 24. 总结

通用牌组的核心价值不是让所有角色变得一样，而是：

> **为每个角色提供江湖中的共同语言，同时让他们用自己的方式理解这些牌。**

同一张“静心”：

- 貂蝉可能用它从魅惑杀局中短暂回神；
- 蔡文姬可能用它延续清音防御；
- 赵云可能用它稳住长枪守护本心；
- 诸葛亮可能用它等待下一回合布局。

同一张“墨刃”：

- 貂蝉可能将它化为心魔魅影；
- 蔡文姬可能把它视为悲音失控；
- 赵云可能在绝境中借它破阵；
- 诸葛亮可能把它当作改命代价。

因此，通用牌组应承担三重功能：

1. 补足战斗基础；
2. 制造构筑变化；
3. 让世界观中的江湖、心境与墨灾进入每一场战斗。

最终目标是让玩家感受到：

> **每一张通用牌都不是冰冷数值，而是一种江湖选择。**
