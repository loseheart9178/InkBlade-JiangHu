# 云水江湖：奇遇系统美术资产优化交付文档 (V1.1)

## 1. 项目概况
本项目已完成“墨录图鉴”中奇遇残页的补全。目前游戏中共有 **45个** 奇遇事件（含角色专属及通用事件），所有事件均已关联至图鉴系统，并支持分章节收录与查看。

## 2. 资产交付规格 (Delivery Specs)
*   **存放位置**：项目根目录下的 `public/assets/generated/events/`
*   **命名规范**：`event_[event_id].png` (例如: `event_black_rain_ferry.png`)
*   **分辨率**：1280 × 720 px (16:9)
*   **视觉风格**：高对比度水墨 (High Contrast Ink Wash)，写意构图，局部朱砂红/青黛色点缀。

## 4. 交付流程建议
1.  **生成**：使用 Midjourney 或 Stable Diffusion 生成上述 45 张图片。
2.  **后期**：统一调整色阶，确保黑色足够深，白色保留纸张质感。
3.  **部署**：将文件放入 `public/assets/generated/events/` 目录下。
4.  **生效**：当前系统已支持图鉴收录逻辑，后续可在 UI 代码中关联具体路径。
