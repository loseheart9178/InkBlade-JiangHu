# Combat UI Kit Gate9: Result / Final Choice / Chapter Transition

## Goal

Bring chapter transition, final choice, defeat/victory result, and completed-run summary surfaces into the same UI kit language established by Gates 2-8.

## Scope

- Chapter reward and boss reward transition shells.
- Transition hero, cinematic rail, progress, spoils dossier, and choice sections.
- Final choice ritual and ending choice cards.
- Defeat/victory result dossier and primary restart action.
- Completed-run summary dossier and scroll region.

## Non-Goals

- No gameplay logic changes.
- No ending eligibility, reward, recovery, profile, or storage logic changes.
- No title, map, compendium, event, rest, reward, or shop redesign.
- No bitmap asset generation in this gate.

## Acceptance

- Focused Playwright coverage proves the transition/result surfaces expose Gate9 UI kit hooks.
- Existing final boss route, defeat result, and first chapter transition flows still pass.
- Full verification remains green after merge.
