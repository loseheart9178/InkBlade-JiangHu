## 2. 角色一句话定位

**诸葛亮是一名通过观星操控牌序、布置阵法改变战局，并在关键时刻借风爆发的高策略角色。**

他不是依赖直接数值压制，而是通过预测、安排、延迟收益和阵法联动让战斗进入自己设计好的局面。

## 3. 服装体态设定

### 3.1 体态气质

- 身形清瘦挺拔，气质沉稳。
- 动作克制，以执扇、观星、落子、布阵为主。
- 表情淡然，眼神像在看更远处的因果。
- 战斗时不应有夸张肢体动作，而是让阵法、风、星象替他行动。
- 整体气质应是“智者的压迫感”。

### 3.2 服装关键词

- 月白、青灰、淡金、墨蓝为主色。
- 宽袍大袖，带道家与士人气质。
- 羽扇是核心标志，应设计精致但不浮夸。
- 背后可有星盘、八阵图、竹简、符纸。
- 衣纹可使用云纹、星轨、八卦、阵纹。
- 不做仙人化，而是“半人半谋”的历史幻想感。

### 3.3 视觉符号

| 元素 | 含义 |
|---|---|
| 羽扇 | 谋略、从容、控场 |
| 星盘 | 观测命数 |
| 八阵图 | 空间、秩序、陷阱 |
| 东风 | 借势、爆发 |
| 竹简 | 记录与推演 |
| 棋子 | 布局、牺牲、选择 |

## 5. 背景故事

诸葛亮比任何人都更早察觉到墨灾的异常。

起初，只是星象错位。  
后来，风向违背节气，棋局中的死子自行复位，竹简上的文字在夜里改写。  
他翻阅旧史，却发现某些本应确定的结局出现了多个版本。

墨书阁向他发来邀请，称墨灾并非灾难，而是“改写败局的机会”。

诸葛亮没有立刻拒绝。

因为他比任何人都清楚：有些败局，如果能重来一次，也许真的可以少死很多人。

但他也知道，一旦人开始相信自己有资格替天下改写命运，便会比任何暴君都危险。

他踏入云水江湖，是为了查清墨书真相，也是为了审问自己：

> “若天命可改，我是否有权落下那一子？”

## 7. 战斗机制总览

### 7.1 核心资源：筹策

**筹策**代表诸葛亮在战斗中积累的谋略资源。

获得方式：

- 使用观星牌：+1 筹策
- 布置阵法：+1 筹策
- 回合结束时若未受到伤害：+1 筹策
- 成功触发敌人意图反制：+1 筹策

消耗方式：

- 调整抽牌顺序
- 强化阵法
- 借风爆发
- 取消或削弱敌人意图

基础规则建议：

```text
筹策上限：9
战斗开始：1
回合结束：全部保留
每回合最多通过卡牌获得5点筹策
```

---

### 7.2 核心机制：观星

**观星**让诸葛亮查看并调整抽牌堆顶部的卡牌。

基础规则：

```text
观星 X：查看抽牌堆顶 X 张牌，可以选择任意数量置于牌堆底，其余按任意顺序放回牌堆顶。
```

观星是诸葛亮最重要的“控牌”能力，能提高构筑稳定性。

---

### 7.3 核心机制：阵法

阵法是持续性的战场效果。  
同一时间最多存在一个主阵法，部分遗物可扩展为两个。

基础阵法类型：

| 阵法 | 效果 |
|---|---|
| 八阵 | 回合结束时获得护甲 |
| 火阵 | 每当敌人受到状态伤害，追加火伤 |
| 风阵 | 每回合第一次打出技能牌时，抽1张牌 |
| 石阵 | 敌人多段攻击伤害降低 |
| 空城阵 | 若本回合不打出攻击牌，敌人攻击降低 |

阵法强调“提前布置，后续收益”。

---

### 7.4 核心机制：借风

借风是诸葛亮的爆发机制。

基础规则建议：

```text
当筹策达到6点以上，可触发借风相关卡牌。
借风通常消耗筹策，产生强力全体伤害、抽牌、回能或阵法强化。
```

借风要有强仪式感，像“布局终于完成”。

## 11. 构筑流派

## 11.1 观星控牌流

### 核心思路

通过观星与检索稳定抽到关键牌，实现高一致性循环。

### 核心卡牌

- 观星
- 锦囊
- 隆中对
- 改命一子
- 风刃

### 核心遗物

| 遗物 | 效果 |
|---|---|
| 星盘残片 | 每场战斗第一次观星时，抽1张牌 |
| 竹简筹 | 观星时每放回2张牌，获得1点护甲 |
| 棋子玉盒 | 每回合第一次调整牌序后，获得1筹策 |

### 优点

- 稳定性极高。
- 能支撑复杂构筑。
- 高手上限很高。

### 缺点

- 输出偏慢。
- 操作时间长。
- 新手难以判断最优牌序。

---

## 11.2 阵法控场流

### 核心思路

通过阵法获得持续收益，让战场逐渐进入自己的节奏。

### 核心卡牌

- 小八阵
- 布阵
- 八阵图
- 空城
- 草船借箭

### 核心遗物

| 遗物 | 效果 |
|---|---|
| 八卦阵旗 | 可同时存在2个阵法 |
| 石兵符 | 阵法触发时，随机敌人获得1层虚弱 |
| 风后图 | 阵法持续时间结束时，获得1筹策 |

### 优点

- 长线战斗强。
- 防御与控场优秀。
- 与世界观高度契合。

### 缺点

- 启动慢。
- 阵法被驱散时损失大。
- 需要清晰 UI 展示。

---

## 11.3 借风爆发流

### 核心思路

积累筹策后，通过借风系卡牌打出高爆发。

### 核心卡牌

- 东风起
- 借东风
- 连环火
- 火计
- 谋定后动

### 核心遗物

| 遗物 | 效果 |
|---|---|
| 东风羽 | 第一次借风消耗筹策减少2 |
| 赤壁火种 | 对灼烧敌人使用借风牌，伤害+25% |
| 七星灯 | 筹策达到上限时，获得1点能量 |

### 优点

- 爆发强。
- 终局爽感明显。
- 视觉演出空间大。

### 缺点

- 前期较弱。
- 害怕资源被打断。
- 对能量和抽牌要求高。

---

## 11.4 空城反制流

### 核心思路

通过少打或不打攻击牌换取防御、反制、下回合爆发。

### 核心卡牌

- 空城
- 谋定后动
- 锦囊
- 草船借箭
- 天命可违

### 核心遗物

| 遗物 | 效果 |
|---|---|
| 空城琴案 | 若本回合未打出攻击牌，回合结束时对随机敌人造成6点伤害 |
| 无兵城门 | 空城效果触发时，获得1筹策 |
| 羽扇纶巾 | 每场战斗第一次不打攻击牌的回合，获得2点能量 |

### 优点

- 非常独特。
- 策略感强。
- 适合高阶玩家。

### 缺点

- 容易过于被动。
- 需要敌人意图配合。
- 平衡难度较高。

## 13. 角色专属事件

### 13.1 星象错位

夜空中，北斗倒悬。  
星光落在水面，却映出另一种结局。

选择：

1. **推演星图**：获得卡牌“隆中对”，失去4点生命。
2. **逆转一星**：获得心法“观星术”，加入诅咒“天疑”。
3. **合上星盘**：提升宁心境，获得金币。

---

### 13.2 墨书阁来信

一封没有落款的书信放在案上。  
信中写道：“败局并非不可重来。”

选择：

1. **赴约一谈**：进入特殊事件链，可能获得“天命可违”。
2. **焚毁来信**：删除一张牌，提升悟心境。
3. **留作证物**：获得遗物“竹简筹”。

## 15. 数值设计注意事项

1. 诸葛亮的控牌能力非常强，基础数值必须偏低。
2. 观星会显著提升牌组稳定性，不能附带过多免费抽牌。
3. 阵法 UI 必须清楚，否则玩家难以理解持续效果。
4. 借风爆发要有充足前置成本。
5. 空城流不应鼓励无操作，而是鼓励有意识地延迟收益。
6. 筹策可保留，因此上限与获得速度是核心平衡点。
7. 高阶角色可以复杂，但初始几场必须给玩家明确教学。

# 角色总结

诸葛亮的核心体验应是：

> **真正的胜利不是打出最强的一张牌，而是让下一回合也在我的推演之中。**

他的战斗体验关键词是：

**预判、控牌、布阵、借势、延迟、改命。**
