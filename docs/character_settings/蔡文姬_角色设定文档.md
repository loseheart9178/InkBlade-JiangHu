## 2. 角色一句话定位

**蔡文姬是一名通过琴音积累音律、用余韵延续效果，并在关键回合奏出终曲的节奏型防御角色。**

她的核心体验不是立即爆发，而是像谱曲一样安排回合节奏：起音、承音、转音、终曲。

## 3. 服装体态设定

### 3.1 体态气质

- 体态端庄柔和，站姿稳定，动作幅度不大。
- 战斗姿态以抚琴、拨弦、闭目听音为主。
- 气质有书卷气和悲悯感，但不能显得软弱。
- 目光安静、清澈，像能听见常人听不见的声音。
- 身边常有琴音化成的墨色波纹与青绿色符文。

### 3.2 服装关键词

- 青白、月白、浅绿为主色。
- 服装形制偏汉魏风格，宽袖、披帛、层叠裙摆。
- 衣纹可有竹叶、流水、五音符号、古谱纹样。
- 腰间可佩小型玉笛、残谱、药囊。
- 古琴既是武器也是情感核心，应高度设计化。
- 琴弦可在技能释放时延展为光线或墨线。

### 3.3 视觉符号

| 元素 | 含义 |
|---|---|
| 古琴 | 记忆、哀思、渡魂 |
| 竹叶 | 清雅、坚韧 |
| 残谱 | 破碎的过去 |
| 青光 | 净化与安抚 |
| 水波 | 余韵、情绪扩散 |
| 断弦 | 痛苦、爆发、诀别 |

## 5. 背景故事

蔡文姬一生听过太多离别的声音。

战火烧过城池时，哭声像风一样穿过街巷；  
远离故土时，马蹄声像无数次回头却不能停下；  
归来之后，她以为自己终于回到人间，却发现记忆并不会因为归来而停止疼痛。

墨灾爆发后，琴弦第一次在无人拨动时自鸣。  
她听见无数亡魂在黑雨中呼唤，有人求归乡，有人求复仇，有人只想让自己的名字不要被忘记。

于是她带着古琴踏入云水江湖。

她并不认为自己是救世之人。  
她只是无法装作没有听见那些声音。

> “若悲声不能止息，我便以此弦，为他们奏到天明。”

## 7. 战斗机制总览

### 7.1 核心资源：音律

**音律**代表蔡文姬在战斗中通过琴音积累的节奏力量。

获得方式：

- 打出琴音牌：+1 音律
- 净化负面状态：+1 音律
- 触发余韵：部分卡牌 +1 音律
- 打出五音牌：获得对应音符

消耗方式：

- 强化琴音伤害
- 转化为护甲
- 触发终曲
- 清除杂音

基础规则建议：

```text
音律上限：10
战斗开始：0
回合结束：全部保留
部分强力卡会消耗大量音律
```

---

### 7.2 核心机制：余韵

**余韵**代表上一回合的琴音仍在战场中回响。

基础规则建议：

```text
当玩家打出带有“余韵”的卡牌时，该卡牌的一部分效果会在下回合开始时再次触发。
同一时间最多保存3个余韵效果。
余韵触发不消耗能量。
```

示例：

- 清心曲：本回合获得护甲，下回合开始再获得少量护甲。
- 幽兰操：本回合施加虚弱，下回合开始再次施加1层。
- 胡笳拍：本回合造成伤害，下回合开始造成余音伤害。

---

### 7.3 关键副机制：五音

五音包括：宫、商、角、徵、羽。

每种音符由对应卡牌产生。  
集齐五音后，可触发“终曲”。

基础规则建议：

```text
五音记录在战斗内保留。
每集齐一次“宫商角徵羽”，获得1层“终曲”。
终曲可用于强化稀有卡或释放强力技能。
```

---

### 7.4 负面机制：杂音

杂音是敌人或部分高风险卡牌加入的负面牌。

```text
杂音：状态牌。抽到时不可主动打出。回合结束时若仍在手牌中，失去1点音律。
```

蔡文姬有许多净化杂音并转化收益的卡牌。

## 11. 构筑流派

## 11.1 余韵防御流

### 核心思路

通过大量带余韵的防御和控制牌，形成跨回合收益。

### 核心卡牌

- 清心曲
- 幽兰操
- 余音绕梁
- 清音心经
- 胡笳残谱

### 核心遗物

| 遗物 | 效果 |
|---|---|
| 回声玉 | 每回合第一次余韵触发时，获得2点护甲 |
| 旧琴匣 | 余韵效果提高25% |
| 青灯谱架 | 每场战斗开始时，随机一张琴音牌获得保留 |

### 优点

- 防御稳定。
- 对长线战斗强。
- 操作节奏清晰。

### 缺点

- 爆发较慢。
- 害怕高强度短期爆发敌人。
- 对卡牌顺序有要求。

---

## 11.2 五音终曲流

### 核心思路

通过宫、商、角、徵、羽积累终曲层数，最终释放高额爆发。

### 核心卡牌

- 宫音
- 商音
- 五音调息
- 五音归一
- 绝响
- 悲声入阵

### 核心遗物

| 遗物 | 效果 |
|---|---|
| 五色琴弦 | 每场战斗开始时获得一个随机五音 |
| 朱纹谱页 | 集齐五音时，获得1点能量 |
| 空山回响 | 释放终曲后，保留2音律 |

### 优点

- 成型后爆发极高。
- 很有“谱曲完成”的仪式感。
- 适合 Boss 战。

### 缺点

- 前期启动慢。
- 需要足够抽牌与检索。
- 对新手稍复杂。

---

## 11.3 净化转化流

### 核心思路

将负面状态牌、杂音、诅咒等不利因素转化为护甲、音律和伤害。

### 核心卡牌

- 净尘音
- 清心曲
- 清音心经
- 悲声不绝
- 归雁

### 核心遗物

| 遗物 | 效果 |
|---|---|
| 洗心铃 | 每净化一张状态牌，造成3点随机伤害 |
| 白玉拨片 | 净化状态牌时，有50%概率抽1张牌 |
| 无尘琴布 | 战斗开始时，将1张随机负面牌转化为清心曲 |

### 优点

- 对状态牌敌人强。
- 防御能力优秀。
- 构筑辨识度高。

### 缺点

- 没有负面牌时收益下降。
- 对纯攻击敌人较弱。
- 需要配合敌人与事件设计。

## 13. 角色专属事件

### 13.1 荒寺夜琴

雨落荒寺。  
无人抚琴，琴却自鸣。

选择：

1. **静听一夜**：失去6点生命，获得卡牌“幽兰操”。
2. **拨弦相和**：获得2点最大生命，获得1张“杂音”。
3. **焚香净音**：删除一张状态牌或诅咒牌。

---

### 13.2 断弦残谱

你拾起一页残谱。  
上面的每一个音符都像旧日哭声。

选择：

1. **续写残谱**：获得心法“胡笳残谱”。
2. **折起收好**：获得遗物“朱纹谱页”。
3. **撕去末章**：获得卡牌“断弦”，提升怒心境。

## 15. 数值设计注意事项

1. 音律可保留，因此上限和消耗数值必须谨慎。
2. 余韵是跨回合收益，不能同时提供过高防御和过高伤害。
3. 净化转化流需要敌人和事件提供足够状态牌，否则构筑不成立。
4. 五音终曲流要保证玩家有检索音符的手段。
5. 蔡文姬不能被设计成单纯治疗角色，应突出节奏规划。
6. 杂音不应只惩罚玩家，也要成为可利用资源。

# 角色总结

蔡文姬的核心体验应是：

> **我不是在打牌，我是在一回合一回合谱出一首能渡过苦难的曲子。**

她的战斗体验关键词是：

**清雅、节奏、防守、净化、余韵、终曲。**
